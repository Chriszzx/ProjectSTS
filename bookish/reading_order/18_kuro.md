# 黒の章

一枚の扉を前にして緊張している。
学校に来たのは何日ぶりのことだろう。
まずは長期に休んだことを説明する為に職員室へと足を運び、それから少し遅れて教室の前へとやって来た。
廊下に人の影はない。
既にほとんどの生徒が教室に集まっているのだろう。
扉に手をかけるのを迷っている。
ああ、早くこの中に入らなければ。
懐かしい。
扉を開けた先の光景にそう思った。
教室は一瞬にして静まりかえる。
居心地の悪さに目を逸らし、挨拶の一つでもかけようとした瞬間だった。
宮沢 夏帆: 「紗夜ちゃん！！」
私の名を呼ぶ懐かしい声。
勢いよく、飛びつく。
宮沢 夏帆: 「心配したんだよぉ～！！」
遠野　紗夜: 「……ごめんなさい」
宮沢 夏帆: 「謝っても許さない！」
私の両腕を掴み、きっと勢いよく睨む。
しかし、長くは続かず、すぐに泣きそうな顔へと変わり、ずりずり下へと滑り落ちて行った。
宮沢 夏帆: 「……ほんとにいなくなっちゃうかもって思ったんだからぁ……」
夏帆はその場に頭を抱え蹲る。
華奢な肩が僅かに揺れている。
泣いてくれているのだろうか。こんな私の為に。
担任教師: 「何をやっているんですか。早く席に戻りなさい」
いつの間にいたのだろうか、背後に担任が立っていた。
私は夏帆を促す為、肩へ手を伸ばし、とんとんと軽く叩いた。
宮沢 夏帆: 「う～……」
立ち上がった夏帆の手を取り、席へと移動する。
頬杖をついた夏目君が私達を見て面白そうに笑う。
夏目 悠希: 「バーカ」
宮沢 夏帆: 「べ～っだ！！」
久方ぶりの席につき、前を向くと、他の生徒達がちらちらとこちらに視線を向けているのが見て取れた。
一息吐き、椅子に座り直すと、私はそれらを無視し意識を声だけに集中させた。
宮沢 夏帆: 「紗夜ちゃ～ん！！　もう！　もう！！　会いたかったよぉ～！！」
遠野　紗夜: 「夏帆……」
挨拶が終わった途端、夏帆は再び私に飛びついた。
椅子に座ったままの私をぎゅうと上から抱きしめる。
遠野　紗夜: 「こほっ」
宮沢 夏帆: 「風邪？　だ、大丈夫～！？」
遠野　紗夜: 「……大丈夫です」
夏目 悠希: 「伝染すなよ」
遠野　紗夜: 「ええ。ご心配には及びません」
宮沢 夏帆: 「そっかぁ……。風邪かぁ～……。あ、だから電話しても出なかったとか～！？」
遠野　紗夜: 「……ええ」
電話。
いつ来ていたのだろう。
記憶を辿ろうとするけれど、ここ数日の記憶はひどく曖昧で思い出すことが出来ない。
風邪もいつの間にひいていたのだろう。
抱きしめられながら、そんなことをぼんやりと考えていると、
宮沢 夏帆: 「……ねえ、紗夜ちゃん。ごめんね？」
遠野　紗夜: 「夏帆？」
私から手を離した夏帆は、とても悲しそうな顔をしている。
謝罪の意味を問い質そうとすると、鐘が鳴る。
夏帆は自分の席へと戻ると、今度は笑顔で私に向かって手を振る。
『ごめんね』
夏帆は気にしてるのだろうか。
自分に新しい友人が出来たということを私への裏切りとでも思ったのかもしれない。
ということは、つまり彼女は真実を知っていたことになる。
今まで気付かなかった私自身に鼻で笑う。
彼女は謝る必要などない。
私に彼女の行動を制限する権利などないし、彼女は彼女なのだから。
むしろ、謝るのならば私の方だ。
『ごめんなさい』

日生 光: 「お、本当だ。紗夜ちゃんがいる。おーい」
聞き覚えのある声。
顔を向けると、廊下側の窓から見覚えのある姿が覗いている。

遠野　紗夜: 「お弁当、楽しみにしています」

遠野　紗夜: 「あ……はい。何とか」
桐島 七葵: 「そうか。良かったな」
遠野　紗夜: 「ありがとうございます」
ふと、会話が途切れ、桐島先輩は思い出したかの様に言う。
桐島 七葵: 「そういえば、お前の兄さん、十夜さんは家にいたのか？」
遠野　紗夜: 「兄さんがどうか致しましたか？」
桐島 七葵: 「いや、昨日お前の家に訪ねたんだが、誰も出なくてな」
遠野　紗夜: 「そうなのですか？　……すみません。休みの間はずっと眠っていたものですから……」
桐島 七葵: 「別に。遠野が大丈夫なら良いんだ」
そう言って、私の頭をくしゃっと撫でる。
遠野　紗夜: 「…………」
乱れた髪も直さず、黙って見つめていると怪訝そうな顔をする。
桐島 七葵: 「どうした？」
遠野　紗夜: 「いえ……」
桐島先輩は変わらずに見える。
千代さんが消えた。
桐島先輩にとって千代さんは大切な友人だったはず。
大切な人が消えて、桐島先輩は悲しくはないのだろうか？
そこまで考えて即座に否定する。
悲しくないはずなどない。
辛くないはずがないのだと。
人間はどうして表面的なものしか見えないのだろう。
言葉や表情だけでは読み取ることの出来ないそれを物語を読む様にして理解出来れば良いのに。
遠野　紗夜: 「……こほっ」
日生 光: 「大丈夫？」
遠野　紗夜: 「……ええ」
桐島 七葵: 「おい、そろそろ移動しないとまずいぞ」
日生 光: 「それは確かに。それじゃあね、紗夜ちゃん」
桐島 七葵: 「また後でな」
遠野　紗夜: 「はい」
宮沢 夏帆: 「ばいば～い」

昼休みになったら会えるというのに、わざわざ休み時間に見に来てくれたのか。
宮沢 夏帆: 「先輩達もずっと心配してたんだよ～」
遠野　紗夜: 「そうなのですか？」
宮沢 夏帆: 「うん」
遠野　紗夜: 「そうですか……」
そんな素っ気のない言葉しか出てこなかった。
私はしばらく先輩達がいなくなった廊下を眺め遣った。

私を真っ直ぐ見つめる眸。
私は視線を逸らしてしまう。
鞄につけた白い花に目がいった。
そんな意図を汲み取ったのかは分からないけれど、夏帆は私に向かってにっこりと微笑んだ。
宮沢 夏帆: 「……明日も明後日もずっとずーっと一緒に帰るんだよ？　勿論、そうじゃない日もあるけどねっ」
ぴょこんと跳ねる様にその場をターンして、振り向きざまに小さく手を振る。
宮沢 夏帆: 「ばいばいっ！」
小走りに去っていく彼女の背中をいつまでも見つめていた。
遠野　紗夜: 「こほっ」
私は時計塔を見上げる。
いつもいつもここに来る度に見上げていた。
その度に何故この時計は止まっているのだろうと不思議に思っていた。
けれど、その疑問はいつになっても晴れることはなく、この時計は永遠に止まったままだと思っていた。
ゆっくりと視線を下ろす。
気付くと、周りに人の姿がなくなっていた。
蒼: 「この時計は何故止まっているのか」
白のローブを身に纏い、フードを目深に被った、私のよく知る存在。
蒼: 「お前には分かるか？　紗夜」
彼は私に問いかける。
遠野　紗夜: 「……いいえ。私には分かりません」
お決まりの会話。
ここで出会う度に同じことを繰り返している気がする。
蒼: 「ここでお前と会うたびに同じ様な会話を繰り返しているな」
目を大きく見開く。
蒼: 「何を驚いている」
遠野　紗夜: 「いえ……私も同じことを思っていたので」
蒼: 「そうか。では、たまには違う言葉でお前に訊ねよう」
遠野　紗夜: 「？」
蒼: 「何故、この時計の針はこの時間に止まっているとお前は思う？」
遠野　紗夜: 「それは……分かりません」
蒼: 「だろうな」
私は顔を顰める。
何だろう？　いつもの彼と雰囲気が違う気がするのは。
蒼: 「お前と初めて会ったのもこの時計塔の前だったな」
遠野　紗夜: 「……はい」
蒼: 「お前と出会ってから、一つ季節を跨いでしまった」
蒼: 「それは時が進んでいるということだ。時は進むものだからな」
遠野　紗夜: 「？　そうですね」
蒼: 「紗夜」
一歩、私に近づく。
また、一歩。
私と彼との距離が近づき、彼の蒼い眸がはっきりと見える位置まで近づいた。
私を黙ったまま見下ろしている。
まるで、眸の奥を覗かれている様。
蒼: 「紗夜。おいで」
私に向かって伸ばされた手。
私は止まったまま動かない。
いつもと違う彼の雰囲気にのまれたのか、それとも本当に時が止まってしまったのか。
その手が触れる間際、私は彼の傍から離される。
遠野　紗夜: 「っ！」
遠野 十夜: 「やあ、蒼。紗夜に何か用かい？」
蒼: 「十夜」
蒼い眸は益々深い色をさせる。
こちらが驚くくらい、彼から強い感情を感じた。
兄は私を隠す様に立つ。
遠野　紗夜: 「兄さん？」
背中から窺うように見上げた兄の横顔は、とても険しい。
一体、どうしたというのだろう？
蒼: 「お前はいつまで気付かないふりをしている？」
遠野 十夜: 「さあ、何を言っているのかさっぱり分からないね」
蒼: 「…………」
遠野 十夜: 「…………」
互いに睨み合った後、兄は蒼に背を向けると
遠野 十夜: 「行こう、紗夜」
そう、笑顔で
遠野　紗夜: 「あっ」

遠野　紗夜: 「…………」

ふっと笑い、次には目を伏せた。
遠野　紗夜: 「けれど、学校には兄さんがいない」
遠野 十夜: 「…………」
私は膝の上から起きあがると、体を向き変え、今度は兄の足の間に自分の足を絡めさせる様に座り直した。
