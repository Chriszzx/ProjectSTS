# 序章 · 实验翻译（第三稿）

> **原则**：保留日文原文的连续阅读感——即中文读者读起来的感觉，应与日文读者读原文时同样顺畅自然。  
> **保留二稿的**：角色语体区分（纱夜敬体、十夜语尾、苍短句）。  
> **放弃二稿的**：逐逗号对标点位的机械对齐。中文按中文自己的呼吸来断句。  
> **比现行版多做的**：不补连接词（却/但/于是）、不合并短句、角色声音有区分度。

## 一、框体故事：死神与旅人少女

```
陽が沈み、夜が来るまでの一瞬の空の色。
それは深く、深く。
蒼色の空。
その下で佇む死神の姿があった。
```

夕阳沉落，夜幕降临之前那一瞬间的天空的颜色。
那是深邃的，深邃的。
苍蓝色的天空。
在那之下，伫立着一个死神的身影。

```
「お前は、私が怖くないのか？」
死神は、臆せず自分の前に立つ旅人を不思議に思った。
```

「你，不怕我吗？」
死神对毫不畏怯地站在自己面前的旅人感到不可思议。

```
「貴方が死神ならば、貴方は私の命を今ここで奪いますか？」
「私が死神であるならば、私はお前の命を奪うのだろう」
「それは困りました。それでは私の旅がここで終わってしまいます」
```

「倘若您是死神，您会在此刻夺去我的性命吗？」
「倘若我是死神，那我大概会夺去你的性命吧。」
「那可就麻烦了。那样的话，我的旅程就要在这里结束了。」

```
旅人である少女に死神は問うた。
「何故、旅をする？」
少女は答えた。
「私は、世界で最も美しい言葉を探しているのです」
```

死神向身为旅人的少女问道。
「为什么要旅行？」
少女答道。
「我在寻找这世上最美的话语。」

```
「美しい言葉、それは詩か？」
「私が求めるのはもっと単純なものです」
「では、愛か」
「いいえ」
死神の言葉に少女は首をふる。
```

「美丽的话语，是指诗吗？」
「我寻求的是更单纯的东西。」
「那么，是爱吗？」
「不。」
听了死神的话，少女摇了摇头。

```
「愛？　平和？　正義？
違う。それらのどれもが私の心を打ち響かせない。
私が探しているものは、唯一であり、絶対であり、永遠であるもの。
世界の全てが認める、
そして共有することが出来る美しい言葉」
```

「爱？和平？正义？
不对。这些没有一个能叩响我的心。
我寻找的，是唯一的、绝对的、永恒的东西。
被全世界所认可，
并且能够共同拥有的，美丽的话语。」

```
「そんなものがこの世にあるというのか？」
「分かりません。だから、私は探すのです。
どこまでもどこまでもどこまでも……この世界全てを巡り、
答えに辿り着くまで」
```

「这世上真的存在那种东西吗？」
「不知道。所以我才会去寻找。
无论多远、多远、多远……巡游这整个世界，
直到抵达答案的那一天。」

## 二、笼中鸟（第一遍朗读）

```
私と彼は一冊の本を読んでいた。
私は甘い声に耳を澄ませながら、目の前の白い紙に書かれた文字を見つめる。
```

我和他一起读着一本书。
我一边倾听着那温柔的声音，一边凝视着眼前白纸上写下的文字。

```
遠野 十夜: 「ここに、一羽の籠の鳥がおりました。何の変哲もない鳥でした。他と比べて特別美しいわけでも、鳴き声が声高いわけでもないただの鳥でした」
```

远野十夜：「这里，有一只笼中鸟。是一只毫不起眼的鸟。不比别的鸟更美，叫声也并不高亢，只是一只普普通通的鸟。」

```
遠野 十夜: 「籠の鳥はよくおなきになります。『籠の外に出たい』籠の鳥の口癖でした」
```

远野十夜：「笼中鸟总爱哭啼。『想出去，到笼子外面去』——这是笼中鸟的口头禅。」

```
淡々とあまり抑揚のない口調は、まるでさざ波のように、私の心を深く、強く、揺り動かす。
もう、何度もこの声を耳にしたはずなのに、未だに心惹かれ、目を細める。
```

那平淡的、几乎毫无起伏的语调，像涟漪一般，深深地、强烈地摇动着我的心。
明明已经听过这个声音无数次了，却仍然被它牵引，微微眯起了眼睛。

## 四、十夜透露新作构想

```

```

远野纱夜：「死神……？」
远野纱夜：「是神话里那种死神吗？就是……收割人命的那种？」

遠野 十夜: 「死神として人から畏れられ、夜に彷徨う」

## 六、纱夜内心独白

```
私はこの時間が何よりも好きだった。

兄の声が物語を紡いで、私がそれを頭の中で想像する。

```

我最喜欢的就是这个时间。
从最喜欢的哥哥那里，听他读最喜欢的书。

遠野 十夜: 「ああ、頼むよ」
