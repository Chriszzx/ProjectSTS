# 《死神与少女 Bookish》术语翻译准则

> 日期：2026-06-08  
> 范围：`bookish/reading_order/*.md` 与 `bookish_zhcn/reading_order/*.md`  
> 用途：供术语回填、后续 LLM 翻译实验和人工审校使用。  

---

## 一、确定译法

| 日文 | 中文固定译法 | 说明 |
|---|---|---|
| `時計塔` | 时钟塔 | 全书空间核心。按原文三字长度对齐，旧译中的“时计塔/钟塔/钟楼”统一为“时钟塔”。 |
| `司書` | 图书管理员 | 不再混用“司书”。说话人标签和叙述均用“图书管理员”。 |
| `物語` | 物语 | 按字面和语境处理。`物語` 可译“物语”，`童話/御伽噺` 译“童话”；中文无自然对应时再调整。 |
| `幻想物語` | 幻想物语 | 不译“幻想故事”。 |
| `籠の鳥` | 《笼中鸟》 | 只固定标题，不强行统一所有 `籠/格子` 意象。 |
| `ユメミルセカイ` | 《梦见世界》 | 第二章书名固定译法。按上下文使用书名号或原文对应的二级引号；普通短语 `夢見る世界` 不机械并入标题。 |
| `秋桜` | 秋樱 | 千代线揭示真名、诗性段落使用。 |
| `コスモス` | 波斯菊 | 揭示前可用“波斯菊”隐藏谜底，不提前暴露“秋樱”。 |
| `桜` | 樱花 | 与 `秋桜` 构成春樱/秋樱对照。 |

## 二、暂不机械替换

### 2.1 `物語`

`物語` 是全书结构词，但不能把所有“故事”机械替换为“物语”。  
处理顺序：

1. 原文明确是标题、结构概念、叙事层级时，可用“物语”。
2. 原文是普通 `話/話す/お話` 或中文必须自然表达时，保留“故事/说话/谈话”。
3. `童話/御伽噺` 固定为“童话”。

### 2.2 `言葉`

`言葉` 不做全局机械替换。日语里的 `言葉` 覆盖“话、话语、词、语言、表达、言辞”等多种中文表达，必须按上下文决定。

候选策略：

1. 核心母题 `美しい言葉 / 世界で一番美しい言葉` 用“美丽的话语 / 世界上最美的话语”。“言语”偏书面，除非刻意保留古典或说明腔，不作为默认译法。
2. 某人说的话、某句话、对白内容，优先用“话/那句话/那些话”。
3. 单个 lexical word，优先用“词/词语/这个词”。
4. 实际语言系统、外语、语言不通，保留“语言”。
5. `花言葉` 固定为“花语”。
6. 故事文本中的一句，可用“故事里的话/台词/那句话”。
7. `Depth` 的副标题统一作“深层之语”。

### 2.3 `本物 / 偽物`

`本物/偽物` 不做全局机械替换。上一轮试用“正牌/冒牌”，但中文里“正牌/冒牌”更适合作修饰语或成对标签，不宜无条件当常规名词。

候选策略：

1. 修饰人名/身份时，可用“正牌的日生光 / 冒牌的日生光”。
2. 成对标签或省略修饰对象时，可用“正牌和冒牌的区别”。
3. 独立名词或谓语位置，按语气改为“本人/正主/真的那个/真货/冒牌货/假的那个/冒名顶替者”等。
4. 童话、亲缘、死神等非日生身份轴语境，按中文自然真/假表达处理。
5. `真货/假货` 带商品感，只在口语、轻佻或讽刺语气中考虑。

### 2.4 `蒼 / 蒼色 / 青`

`蒼` 同时是角色名、颜色、天空、眼睛、黄昏/夜色母题，必须结合上下文检查。

处理原则：

1. `蒼色` 与苍、天空、眼睛、书脊、夜色母题相连时优先译“苍色”。
2. `青/青い` 只是普通蓝/青时可译“蓝/青”。
3. 涉及“苍”这个人物命名、眼睛、天空回环的段落，不做无上下文自动替换。

## 三、本轮自动回填边界

可自动回填：

- 时计塔/钟塔/钟楼 -> 时钟塔
- 司书 -> 图书管理员
- 大波斯菊 -> 波斯菊
- 樱花花 -> 樱花
- 秋樱花 -> 秋樱
- 幻想故事 -> 幻想物语
- 标题《笼里的鸟》/《笼中之鸟》 -> 《笼中鸟》

只扫描记录，不自动回填：

- 剩余“故事”是否应为“物语”
- `言葉` 剩余项对应“话/话语/词/语言/台词”等哪一种
- `蒼色/青色/蓝色` 与苍人物母题的对应
- `本物/偽物` 剩余项对应“正牌/冒牌/正主/本人/冒牌货/真货/真的那个/假的那个”等哪一种

## 四、上下文修正边界：`言葉` 与 `本物/偽物`

本轮为这两个语义敏感组新增单独脚本 `scripts/contextual_term_apply.py`，不并入“无需上下文判断”的主术语脚本。

可自动修正：

- `美しい言葉 / 世界で一番美しい言葉` 等主题短语 -> “美丽的话语 / 世界上最美的话语”。
- 山丘约定场景 `世界を彩る言葉` -> “给世界增添色彩的话语”；描绘景色的具体表达可用“词语”。
- `花言葉` -> “花语”；`花に言葉を込める` -> “把话语寄托在花上”。
- 书面腔过重的“言语”在发话/安慰/伤人语境中改为“话语/话”。
- 日生身份轴中修饰人名的 `本物/偽物` -> “正牌/冒牌”。
- 人物身份轴里不自然的“真品/赝品” -> “正牌/冒牌”或“冒牌货”。
- 严肃身份确认中的“本尊” -> “本人”。

继续人工审核：

- 实际语言系统、外语、语言不通、书名《花之语言》等不自动改。
- `真货/假货/正主/本人/冒牌货` 的剩余名词位按说话人语气逐条判断。
